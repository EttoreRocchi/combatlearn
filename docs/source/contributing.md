# Contributing to combatlearn

Thanks for your interest in contributing! Here's how to get started.

## Development Setup

```bash
git clone https://github.com/EttoreRocchi/combatlearn.git
cd combatlearn
pip install -e .[dev]
```

## Running Tests

```bash
pytest
```

## Linting

```bash
ruff check .
```

## Submitting Changes

1. Fork the repository and create a feature branch from `main`.
2. Add tests for any new functionality.
3. Make sure all tests pass and the linter is clean.
4. Open a pull request with a clear title and description of your changes.

## Reporting Issues

Open an issue on [GitHub](https://github.com/EttoreRocchi/combatlearn/issues) with a minimal reproducible example if applicable.
